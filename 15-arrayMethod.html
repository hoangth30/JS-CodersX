ARRAY METHOD

  • Array.from()

    Creates a new, shallow-copied Array instance from an array-like or iterable object.

    - array-like objects (objects with a length property and indexed elements);

    - iterable objects (objects such as Map and Set)

    Return VALUES OF ARRAY

  • Array.isArray()

    Determines the passed values is an array or not

    Return TRUE if the value is ARRAY, otherwise, FALSE

  • Array.of()

    Creates a new array instance from variables number of argument or types

    Return ARRAY INSTANCE

  • Array.concat()

    Used to merge two or more arrays, does not change the existing array

    Return NEW ARRAY

  • Array.slice()

    Creates a new shallow-copied of 1 part from existing array into new array, which is selected from begin to end (not included end). begin/end represents the index of items. The original array is not modified

    + begin: 1. negative index can be used, e.g: slice(-2), takes two last elements; 2. undefined, start from 0; 3. greater than range, return empty array

    + end:

    Return ARRAY CONTAINING EXTRACTED ITEMS

  • Array.splice()

    Changes the contents of an array by removing or replacing existing elements and/or adding new elements in place
